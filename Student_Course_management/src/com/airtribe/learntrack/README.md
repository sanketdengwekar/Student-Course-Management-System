Student & Course Management System

**Features:**
- Student CRUD (Create, Read, Update, Deactivate)
- Course management (Add, List, Toggle status)
- Enrollment system (Enroll, View, Update status)


## Tech Stack
- **Java 21** (JDK)
- **Core Java** (no external libraries)
- **OOP**: Classes, Inheritance, Polymorphism
- **Collections**: ArrayList
